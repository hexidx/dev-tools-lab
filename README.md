# dev-tools-lab

A public sandbox for small developer tools, CLI utilities, validators, parsers, and automation helpers.

## Goals
- Ship small useful tools continuously
- Keep changes safe, testable, and well-scoped
- Let autonomous coding agents work on clearly bounded tasks

## Structure
- `src/dev_tools_lab/` main package
- `tests/` automated tests
- `BACKLOG.md` task queue
- `AGENT.md` rules for autonomous work


## Included utilities
- `dev_tools_lab.line_count.count_lines` for counting lines in text.
